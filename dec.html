🚀 Dev Vision Documentation

Introduction

Dev Vision is a platform designed to help young African developers grow their careers by helping them build professional portfolios and showcase their skills to the world.

Portfolio Creation

1. Login to Dev Vision 2. Go to Portfolio Page 3. Fill Name, Title, About and Skills 4. Click Save Portfolio

Work Upload System

Go to Work Page → Add project links and description → Save. Your work will automatically appear on your portfolio profile.

Analytics System

Dashboard shows: • Portfolio views • Number of works • Visibility ranking score
